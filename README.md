# 🛡️ Enterprise Infrastructure Lab

## Overview

This repository documents my hands-on Enterprise Infrastructure Home Lab, where I build, configure, secure, and document Windows Server environments from the ground up.

The purpose of this project is to simulate a real enterprise IT environment while strengthening my skills in Windows Server administration, Active Directory, networking, automation, and cybersecurity.

Rather than simply following tutorials, I focus on understanding **why** each technology exists, **how** it works, and **how** it is implemented in production environments.

# 🎯 Learning Objectives

By completing this project, I will gain practical experience with:

## Windows Infrastructure

- Deploy Windows Server
- Install and configure Active Directory
- Configure DNS and DHCP
- Design Organizational Units (OUs)
- Create and manage users and groups
- Deploy Group Policy Objects (GPOs)
- Join Windows clients to a domain
- Configure File Servers and NTFS Permissions
- Automate administration using PowerShell

## Cybersecurity

- Secure Active Directory
- Monitor authentication events
- Analyze Windows Event Logs
- Build a Security Information and Event Management (SIEM) lab
- Investigate Active Directory attacks
- Practice incident response
- Implement security hardening best practices
